# MiniSky Visual Identity: AI Brand Prompts

This document contains specialized prompts designed to generate striking, professional brand assets for MiniSky across various AI platforms (Midjourney, DALL-E 3, Adobe Firefly, etc.).

## Brand Aesthetic Profile
- **Concept:** "The Cloud, local and contained."
- **Keywords:** High-Fidelity, Satellite, Orbit, Infrastructure, Modern, Premium, Glassmorphism.
- **Color Palette:** 
    - Primary: **Google Blue** (#4285F4)
    - Secondary: **Slate Dark** (#1A1A1B)
    - Accents: Vibrant Green (#34A853) and Aurora Purple (for a "sky" feel).

---

## 1. The Master Brand Prompt (Logo & Icon)
Use this for the primary logo, X logo, Facebook profile, and general social media icons.

> **Prompt:** "A professional, minimalist logo for a high-tech cloud infrastructure product named 'MiniSky'. The design should feature a sleek, stylized satellite or a cloud enclosed within a perfect circle, representing 'local cloud containment'. Use a clean, modern aesthetic similar to premium tech brands like Stripe or Vercel. Primary color: Google Blue (#4285F4) on a dark slate background. High contrast, vector style, flat design with a subtle 3D glassmorphism effect on the edges. No text, focus on a recognizable mark."

---

## 2. LinkedIn Cover Page (Hero Image)
Designed for the wide aspect ratio of LinkedIn.

> **Prompt:** "A high-resolution LinkedIn banner for a developer tool called MiniSky. The scene depicts a futuristic, dark-themed command center workspace. In the center, a glowing, 3D holographic cloud is floating above a sleek laptop. Intricate network nodes and lines connect the laptop to the cloud. Style: Cinematic, tech-focused, deep blues and purples with vibrant green data stream accents. Bokeh background of abstract code. Atmosphere: Professional, reliable, state-of-the-art. 4k resolution, wide aspect ratio."

---

## 3. X (Twitter) & Facebook Header
A punchier, more energetic version of the brand.

> **Prompt:** "An energetic social media header for MiniSky. Abstract representation of data traveling through a satellite orbit around a stylized 'Localhost' icon. Bold typography in the corner saying 'MiniSky' in a clean sans-serif font like Inter or Roboto. Colors: Vibrant Google Blue and Deep Black. Dynamic light streaks, high-tech infrastructure patterns, minimalist and premium. Clean lines, ultra-modern."

---

## 4. "The High-Fidelity" Visual (Social Media Post Image)
Use this for LinkedIn/X posts to show "Product Power."

> **Prompt:** "A striking 3D isometric illustration of a local server rack that looks like a miniature Google Cloud data center. A glowing 'Satellite' icon is hovering at the top, radiating power. The text 'High-Fidelity' is subtly etched into the side of the rack. Hyper-realistic materials, soft ambient lighting, high tech, premium developer aesthetic. Slate and blue color scheme."

---

## Tips for Best Results:
1. **Aspect Ratios:** If using Midjourney, add `--ar 16:9` for banners or `--ar 1:1` for logos.
2. **Negative Prompts:** If the AI adds unwanted text, use: `--no text, letters, blurry, low resolution, cartoonish`.
3. **Iteration:** If the logo is too complex, add the word "Minimalist" or "Pictogram" to the prompt.
